# whoami — GitHub Pages (DevOps profile)

This is a lightweight, one-page profile designed for hiring managers to quickly understand skills, projects, and certifications.

## Run locally

Open `index.html` in a browser.

If you prefer a local server (optional):

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish on GitHub Pages

### Option A (recommended): use a `username.github.io` repo

1. Create a GitHub repo named `YOUR_GITHUB_USERNAME.github.io`
2. Push these files to the repo root
3. GitHub Pages will serve it automatically at `https://YOUR_GITHUB_USERNAME.github.io`

### Option B: use any repo + Pages from root

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from a branch`
4. Select `main` (or `master`) and `/ (root)`
5. Save

## Content note

This page intentionally avoids listing company names. It focuses on:
- Skills & tools
- Impactful projects
- Certifications
- A short hiring-manager Q&A
